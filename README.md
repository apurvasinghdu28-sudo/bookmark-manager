# 🔖 Bookmark Manager

A simple Bookmark Manager built using HTML, CSS, and JavaScript.

This project allows users to add bookmarks by entering the website name and its URL. Clicking on the created bookmark opens the website in a new browser tab.

## ✨ Features

- Add website bookmarks
- Open bookmarks in a new tab
- Input validation (prevents empty fields)
- Automatically adds `https://` if missing
- Clear all bookmarks from the page
- Simple and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM & Events)

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter the website name and URL.
4. Click **Submit** to add a bookmark.
